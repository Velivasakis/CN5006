1. Για την εργασία χρησιμοποιήθηκε κώδικας PHP και MySQL

    Η Βάση Δεδομένων (mydb) με τους εξής πίνακες:

        users: Εδώ αποθηκεύονται οι Φοιτητές και οι Καθηγητές. Οι κωδικοί τους δεν φαίνονται, είναι κρυπτογραφημένοι
        courses: Τα μαθήματα που φτιάχνουν οι καθηγητές
        course_enrollments: Ποιος φοιτητής παρακολουθεί ποιο μάθημα
        assignments: Οι εργασίες που βάζουν οι καθηγητές
        submissions: Οι απαντήσεις που στέλνουν οι φοιτητές (PDF) και οι βαθμοί τους
        announcements: Οι ανακοινώσεις που φαίνονται στην αρχική
        classes: Ένας βοηθητικός πίνακας για τα τμήματα


        Τα Αρχεία PHP είναι τα εξής:
            db.php: Κάνει τη σύνδεση με τη βάση δεδομένων
            functions.php: Έχει μέσα συναρτήσεις που ελέγχουν αν τα emails και τα ονόματα είναι σωστά γραμμένα
            login.php και register.php: Οι σελίδες για να μπει κάποιος ή να γραφτεί
            index.php: Η κεντρική σελίδα. Μόλις συνδεθείς ελέγχει αν είσαι φοιτητής ή καθηγητής και σου δείχνει τα ανάλογα κουμπιά
            main.php: Η σελίδα που βλέπει όποιος δεν έχει συνδεθεί
            actions_student.php: Εδώ είναι ο κώδικας για τους φοιτητές
            actions_professor.php: Εδώ είναι ο κώδικας για τους καθηγητές



2. Εμφάνιση για να φαίνεται ωραία η σελίδα και να δουλεύει και σε κινητά:

    Bootstrap 5: Για να φτιάξω το μενού και τα κουμπιά εύκολα
    FontAwesome: Για τα εικονίδια
    Leaflet.js: Για να βάλω τον χάρτη στην αρχική σελίδα
